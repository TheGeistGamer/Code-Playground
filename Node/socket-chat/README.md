# 💬 Realtime Chat App (Socket.io)

Aplicación de chat en tiempo real tipo WhatsApp usando WebSockets con Socket.io.

“Chat en tiempo real con múltiples usuarios usando WebSockets”

## 🚀 Tecnologías

- Node.js
- Express
- Socket.io
- CORS
- dotenv

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
npm install

```

## ▶️ Uso

```bash
node index.js
```

Servidor en: `http://localhost:3000`
