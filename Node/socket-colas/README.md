# 🎟️ Queue Management System (Socket.io)

Aplicación en tiempo real para gestión de turnos y tickets (tipo banco) usando Socket.io.

## 🚀 Tecnologías

- Node.js
- Express
- Socket.io
- CORS
- dotenv

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
npm install
```

## ▶️ Uso

```bash
node index.js
```
