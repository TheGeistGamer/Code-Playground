# 🌍 Weather CLI App

Aplicación de consola para consultar la ubicación y el clima en tiempo real desde la terminal.

## 🚀 Tecnologías

- Node.js
- Inquirer
- Axios
- Colors
- dotenv

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
npm install

```
