# 🛒 E-commerce REST API

API REST para gestión de usuarios y funcionalidades básicas de un e-commerce.

## 🚀 Tecnologías

- Node.js
- Express
- MongoDB (Mongoose)
- JWT (jsonwebtoken)
- bcryptjs
- express-validator
- Cloudinary
- express-fileupload
- Google Auth
- CORS
- dotenv

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
npm install

```
