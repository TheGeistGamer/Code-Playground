# 📦 File Upload API with AWS S3

API básica construida con Node.js y Express para subir archivos a AWS S3.

## 🚀 Características

- Subida de archivos a AWS S3
- Manejo de archivos temporales
- Endpoint simple para pruebas
- Estructura modular (helpers para S3)

## 🛠️ Tecnologías

- Node.js
- Express
- express-fileupload
- AWS SDK (S3)
