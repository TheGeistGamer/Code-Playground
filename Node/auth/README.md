# 🔐 Auth API (MongoDB + JWT)

API básica para autenticación de usuarios (registro, login y manejo de usuario) usando Node.js, Express y MongoDB.

## 🚀 Tecnologías

- Node.js
- Express
- MongoDB (Mongoose)
- JWT (jsonwebtoken)
- bcryptjs
- dotenv / env-var

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
